# Server Health Monitor

Automated server health monitoring running on GCP.
Tracks CPU, memory, disk, and Nginx status every hour.
Built with Python + Bash + GitHub Actions.

**Status:** 🔄 In Development — Building Week by Week

## Tech Stack
- Python 3
- Bash Scripting  
- Google Cloud Platform (GCP)
- GitHub Actions (CI/CD)
- Linux (Ubuntu)
